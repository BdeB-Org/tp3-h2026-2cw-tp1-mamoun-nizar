Bienvenue chez MN Sushi

Découvrez une sélection de sushis frais préparés avec soin. Cette application Web permet de gérer le menu, les réservations, les avis, les utilisateurs et les catégories du restaurant.

Voir le menu Réserver une table

Informations techniques

Ce projet utilise HTML, CSS et JavaScript pour le frontend. Les données sont récupérées à partir d’une base de données Oracle avec ORDS.

  • Base de données : Oracle
  • API : Oracle REST Data Services
  • URL de l’API : http://localhost:8080/ords/tp3
  • Tables utilisées : utilisateur, categorie, produit, reservation, avis
